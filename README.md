FRONT-END LAUNCH
1. Run the project in Visual Studio Code
2. Launch Terminal > New Terminal
3. Write "npm i" and Enter
4. Write "npm run dev"
5. Ctrl and click on the link 

BACK-END LAUNCH
1. Launch New Terminal > New Terminal and write "cd back-end"
2. Write "npm run start"
